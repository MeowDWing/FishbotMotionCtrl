# 项目介绍

本项目来源于利用FishBot实现离线固定路线控制与在线控制（在线控制短时间内不做）

## 文件说明

- /src - 主程序代码
- /lib - 功能库函数
- /boards - 主板配置文件

* platformio.ini - 工程配置文件
